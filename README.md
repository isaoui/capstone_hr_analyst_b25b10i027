# Isha: AI-Powered HR Assistant

Asisten HR cerdas yag menggunakan **Gemma 1.1 2B** dan **RAG** untuk analisis resume secara otomatis. Proyek ini dikembangkan oleh saya pribadi **SYAFIQ JUBAIR ISHAMUDDIN** untuk tugas CAPSTONE.

## Tech Stack
* **LLM:** Google Gemma 1.1 2B-IT (4-bit Quantization)
* **Database:** ChromaDB (Vector Search)
* **GPU:** NVIDIA RTX 4050

## Fitur
* Analisis kecocokan skill kandidat secara otomatis.
* Pemberian peringkat dan justifikasi peringkat 1.
* Rekomendasi pertanyaan wawancara untuk HR.
